Introduction

What is GameVault?​

GameVault is a selfhostable source-available gaming platform for DRM-free videogames. It offers a Steam-like browsing and playing experience for you and your community to enjoy the game files stored on your file server in an organized way.

You simply put your game files (installers, binaries, etc.) - from wherever you have them, into a folder and the server detects them, enriches them with metadata and offers a user-friendly interface to easily organize, download, install, and play your games. It even tracks your playtimes and progress.

GameVault consists of two components:

  • A self-hosted backend server served with Docker
  • A client application to use this server
Try it out!

If you don't have a server yet, but want to try it out, you can connect your GameVault Client to our demo server get a feel for how it works.

Features​

Let's take a look at all the main functions of GameVault:

Client Features​

  • 🔎 Browse your server's game collection
  • 📥 Securely download your games from your server
  • ⚙️ Automatically extract and install games
  • 🎮 Launch and play the games on your PC
  • ⏱️ Track playtimes and progress (even when the server is offline)
  • 👥 Compare game progress with other users on the server
  • ✏️ Edit any game metadata
  • 🔒 Manage your users and games with a powerful Admin UI

Server Features​

  • ♻️ Automatic indexer to sync games from the file system into the database
  • 📚 Automatic enrichment of games with metadata, box arts, tags, genres, description and more using various APIs
  • 👥 Multi-User Authentication
  • 🔒 Role-Based Access Control
  • 👨‍👩‍👧‍👦 Parental Control Features
  • ⏱️ Game Progress and Statistics Tracking
  • 🚨 Health Monitoring and Logging
  • 🔎 Full-Text Search, Filters, Sorting, Pagination
  • 🔌 Fully Documented API
  • ⚙️ Highly Configurable
  • ⚙️ Modular Plugin Framework
